Heroic Testimonials lets you add testimonials to your site.

== Description ==

Add customer testimonials, includes a widget for displaying the testimonials in widget areas.


== Installation ==

It's easy to get started

1. Upload `ht-testimonials` to the `/wp-content/plugins/` directory or goto Plugins>Add New and search for Heroic Testimonials.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Goto Appearance > Widgets and drag a Testimonials widget to a widget area.
